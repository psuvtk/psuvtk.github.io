ATF 与 EDKII 的关系

EDKII 与 Das U-Boot 的关系
EDKII 是
Das-Uboot 是嵌入式引导程序的实施标准



非安全启动: Das U-Boot、Das U-Boot + UEFI

安全启动: ATF 、  ATF+EDKII(x)


- AP + SCP
ROM STAGE 加载 RAM STAGE BOOT1 到 SCP SRAM 
RAM STAGE BOOT1 加载 SCP Firmware(M3)
RAM STAGE BOOT1 加载 DDRC Firmware(M7)
RAM STAGE BOOT1 加载 AP RAM STAGE BOOT2
AP RAM STAGE BOOT2 符合 UEFI 标准 PEI - DXE - BDS

- AP + SCP + DDRC
ROM STAGE 加载 RAM STAGE BOOT1 到 SCP SRAM 
RAM STAGE BOOT1 加载 SCP Firmware(M3)
RAM STAGE BOOT1 加载 DDRC Firmware(M7)
RAM STAGE BOOT1 加载 AP RAM STAGE BOOT1
AP RAM STAGE BOOT1 加载 AP RAM STAGE2
AP RAM STAGE BOOT2 符合 UEFI 标准 PEI - DXE - BDS
AP PEI阶段拉起ATF，BL1-BL2-BL32


- 服务器
ROM STAGE 加载 RAM STAGE BOOT1 到 SCP SRAM 
RAM STAGE BOOT1 加载 SCP Firmware(M3)

RAM STAGE BOOT1 加载 AP RAM STAGE BOOT1
AP RAM STAGE BOOT1 加载 AP RAM STAGE2
AP RAM STAGE BOOT2 符合 UEFI 标准 PEI - DXE - BDS
AP PEI阶段拉起ATF，BL1-BL2-BL32
RAM PEI 加载 DDRC Firmware(M7), DDR初始化等
