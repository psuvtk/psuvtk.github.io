published on February 26, 2023
下载Ubuntu镜像 安装Virtual Box 下载内核源码, linux-source 这个包可能和当前的内核版本不匹配？, 所以使用linux-source-5.xx.x包 sudo apt-get install linux-headers-$(uname -r) sudo apt-get install linux-source-5.xx.x cd /usr/src/linux-source-5.19.0 sudo tar jxf /usr/src/linux-source-5.19.0.tar.bz2 依赖安装 sudo apt install build-essential libncurses5-dev libssl-dev flex bison 编译内核 #清除之前编译环境 make mrproper # 编译 sudo make defconfig sudo make -j4 安装基本软件 sudo apt install zsh git curl vim
